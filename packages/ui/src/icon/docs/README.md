# 按钮
Lin UI 致力于给小程序开发者提供愉悦的开发体验。

> 常用的操作按钮

### 主题色按钮

通过设置`type`属性为`default`、success、error、warning来创建不同样式的Button，样式默认为default。

```html
import AccountFilled from '@lin-ui-vue/icons'
<template>
  <div>
    <account-filled>default</account-filled>
  </div>
</template>
```
